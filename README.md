## 🏛️ Sobre o Projeto

**Tower Deep** é um jogo Tower Defense no Roblox ambientado na mitologia grega.  
Este repositório contém o **site oficial** do jogo, desenvolvido como uma página estática hospedada no GitHub Pages.

## 📄 Páginas

| Página | Descrição |
|---|---|
| `index.html` | Página inicial com hero, deuses e screenshots |
| `wiki.html` | Wiki completa do jogo |
| `changelog.html` | Histórico de atualizações |
| `roadmap.html` | Mapa de desenvolvimento |
| `ranking.html` | Ranking de jogadores |
| `votos.html` | Votos em features |
| `enquetes.html` | Enquetes com prazo automático |
| `tier-list.html` | Tier list de torres |
| `tower-balance.html` | Calculadora de balance |
